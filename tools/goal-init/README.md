# @cobusgreyling/goal-init

Scaffolds goal engineering workflows. This CLI initializes a directory with `GOAL.md`, `goal-budget.md`, and relevant skills to support autonomous agents in completing objectives incrementally.

## Usage

```bash
npx @cobusgreyling/goal-init . --tool grok
```

Supported tools:
- `grok` (default)
- `claude`
- `codex`
- `opencode`
